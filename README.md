# Event Booking System

A full-stack event booking application that allows users to browse upcoming events, book and cancel tickets, and receive email confirmations.

Built using **React.js**, **Node.js**, **Express.js**, **MongoDB**, and **Nodemailer**, following RESTful API principles.
